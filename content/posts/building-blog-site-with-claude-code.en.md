I built this blog site together with Claude Code. Here I document the process from tech stack selection to writing.

## Why Blog with Claude Code

I recently started using Claude Code seriously for coding work. Following that thread, I thought "maybe I can build a blog with it too?"

I've tried starting a blog several times before, but always gave up because polishing and fact-checking felt too tedious. Maybe things would be different with a coding agent. With that hope, I set up this blog.

## Tech Stack: Hugo + PaperMod

Decided after consulting with Claude Code.

It said "Hugo for simple blogs, Astro if customization matters more." This time I prioritized getting the blog up and running over customization, so I chose Hugo. Content is in Markdown, and migration costs would be low if I leave it to Claude Code.

For the theme, I discussed requirements like "multilingual support, lightweight, long-term stability" and landed on PaperMod.

But at least, it feels more fun now.
