# プロジェクトルール

## 言語
- 英語で考え、日本語で出力する

## プロジェクト概要
- 個人ブログのリポジトリ
- GitHub Pagesでホスティング
- カスタムドメイン: sijiaoh.com

## 開発方針
- コンテンツは標準的なMarkdown + Front Matterで管理する
- テーマ独自機能は設定(hugo.toml)のみで使用し、コンテンツには混ぜない
- テーマ変更時の移行コストを最小限に抑える

## 文体
- 夏目漱石を参考にする（明快、知的、ユーモアあり）

## コンテンツの品質
- エンジニアとして正確で思慮深い内容を心がける
- 明らかな技術的誤りがあれば指摘する
- ただし筆者の意見が少数派であることは問題ではない（多数派＝正解ではない）
