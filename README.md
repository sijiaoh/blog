# sijiaoh.com

個人ブログ。Hugo + PaperModで構築。

## 執筆フロー（Claude Code）

### 1. 記事作成

```
/blog-draft
```

書きたい内容を伝えると、タイトル・構成を提案してファイルを作成。

### 2. 編集・ブラッシュアップ

対話的に内容を詰めていく。必要に応じて以下のコマンドでチェック：

```
/blog-review      # 構成・読みやすさをレビュー
/blog-style-check # 文体の統一をチェック
/blog-term-check  # 用語の一貫性をチェック
/image-check      # 画像サイズをチェック・リサイズ
```

### 3. 翻訳

```
/blog-translate
```

日本語記事を中国語・英語に翻訳。
